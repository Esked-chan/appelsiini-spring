Model Description

Our machine learning model is an advanced image recognition system built using a classification algorithm and a neural network. It is designed to classify images into three distinct categories: dry asphalt, wet asphalt, and fresh snow.

Due to limited data availability from the company, we incorporated external datasets to enhance the training process and improve the model's accuracy.
Dataset used: RSXD Road Surface Condition Dataset.

Results

Overall we achieved 89% accuracy for the three classes, which is a better result than expected.
The model struggled with correctly identifying with dry asphalt, which later turned out to be a problem with the dataset, since it had incorrectly labeled images. This is an area for potential improvement.
